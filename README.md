# 开发者工具箱

## 项目简介

这是一个现代化的开发者工具箱 Web 应用，提供多种实用的开发工具，全部在浏览器端运行，同时也已打包electron在Windows系统的安装包，请在release下载

## 技术栈

- Vue 3
- Ant Design Vue
- Vue Router

## 安装与运行

```bash
# 克隆项目
git clone https://github.com/yourusername/developer-toolbox.git

# 进入项目目录
cd developer-toolbox

# 安装依赖
pnpm install

# 启动开发服务器
pnpm run serve

# 构建生产版本
pnpm run build

# 构建Tauri 客户端
pnpm run tauri:build
```

  - 编解码工具
  - 加解密工具
  - 时间工具

  - 代码格式化
  - JSON 工具
  - XML 工具
